The popular Scikit-Learn library also contains an implementation of the Multinomial Naive Bayes classifier, so we will compare how our model performs against it.

The Scikit-Learn library doesn't go with text data that well. However, we can covert the Target column to binary numbers as a workaround. Convert ham cells to 0 and spam to 1.

Objectives

In this stage, your program should:

  1. Import MultinomialNB from sklearn.naive_bayes;
  2. Convert training and test datasets to BoW dataframes;
  3. Convert the text entries in the Target column to binary numbers for both training and test BoW dataframes;
  4. Train the MultionomialNB model with the training BoW dataframe;
  5. Make the predictions for the test BoW dataframe.
  6. Compute the Accuracy, Recall, Precision, and F1 scores using sklearn.metrics.
